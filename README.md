# 🔧 Engineer Connect

A modern platform connecting engineering students with real-world problems and innovative solutions.

## ✨ Features

- **Student Dashboard** - Personalized feed for engineering students
- **Problem Solving** - Real company problems to solve and showcase skills
- **Interactive Quizzes** - Test your engineering knowledge
- **Professional UI** - Modern dark theme with teal accents
- **Mobile Responsive** - Optimized for all devices
- **User Profiles** - Comprehensive student and company profiles

## 🚀 Tech Stack

### Frontend
- **React.js** - Modern JavaScript library
- **CSS3** - Advanced styling with glassmorphism effects
- **Responsive Design** - Mobile-first approach

### Backend
- **Node.js** - JavaScript runtime
- **Express.js** - Web application framework
- **MongoDB** - NoSQL database
- **JWT Authentication** - Secure user authentication

## 🛠️ Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn
- MongoDB

### Installation

1. Clone the repository
```bash
git clone https://github.com/yourusername/engineer-connect.git
cd engineer-connect
```

2. Install frontend dependencies
```bash
cd engineer_connect-app
npm install
```

3. Install backend dependencies
```bash
cd ../server
npm install
```

4. Start the development servers

Backend:
```bash
cd server
npm start
```

Frontend:
```bash
cd engineer_connect-app
npm start
```
